# Customer_Shopping_Behavior
Project Overview:
              This project focuses on analyzing customer shopping behavior using retail transaction data. The goal is to uncover patterns in purchasing habits, identify customer segments, and generate actionable insights to improve business decisions related to marketing, customer engagement, and sales.

Tools & Technologies:
    Python (Pandas, NumPy)
    SQL (PostgreSQL)
    Power BI
    
Workflow:
    Cleaned and prepared data using Python
    Performed business analysis using SQL
    Built an interactive dashboard in Power BI

Key Insights:
    Loyal customers contribute the highest revenue
    Discounts strongly influence purchase behavior
    Some products are highly discount-dependent
    Subscription status impacts customer spending
    Revenue varies across different age groups

Outcome:
    Provided data-driven recommendations to improve customer retention, optimize discounts, and target high-value customer segments.
